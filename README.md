# Student Management System

A complete dynamic web application built with PHP, MySQL, HTML, CSS, and JavaScript for managing student records, courses, and academic reports.

## Project Structure

```
/project-root
   /assets
      /css
      /js
      /images
   /config
   /includes
   /php
   /reports
   /views
   /docs
   index.php
   login.php
   dashboard.php
   register.php
```

## Features

- User Authentication (Login/Logout with Sessions)
- Student CRUD Operations
- Course Management
- Academic Reports (3 different report types)
- Responsive UI Design
- Secure Database Operations (Prepared Statements)

## Installation

1. Import the SQL database schema from `/config/database.sql`
2. Configure database connection in `/config/db.php`
3. Set up web server (Apache/Nginx) with PHP 7.4+
4. Access the application through your web browser

## Documentation

Complete project documentation is available in the `/docs` folder.
